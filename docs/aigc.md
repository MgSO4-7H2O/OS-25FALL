# AIGC使用

本课程鼓励AIGC的应用，可以在实验环境中自由使用相关工具解决问题，但是同学们产出的代码和实验报告仍受到实验规范的限制

!!! tip "其实任何情况下都不建议直接采纳AI吐出来的资料，很容易出现实际细节写错的情形；最好是在检索到相关用法后，自己再去搜索确认RISCV手册或者Linux代码仓库中的具体描述，理解实现再开始coding“

**建议使用AIGC的场景**

- 开发环境配置
- 资料查询
- 解释命令和代码的参数
- 帮助编写shell命令
- 配置VSCode调试命令
